# FEATURES I ADDED IN THE FLASK TODO LIST APPLICATION from (https://github.com/Ishan2608/Flask-Todo-List)

1. Delete Option
    -Users can delete mistakenly created task.
    -They can delete task they don't need anymore.
    *If a task is no longer needed, users should be able to remove it. This keeps the task list clean and relevant.*

2. Edit Option
    -Users can edit their created task.
    *Sometimes users want to change the task description later. Allowing them to click “Edit” and update the text improves flexibility.*

3. Task Category Option
    -Users can assign category on their task to easily remember the purpose of their task
    *Allow users to assign a category like “Work,” “House,” or “Shopping.”*

4. Automatically Task Priority (according to date deadline)
    -Users can priority their much needed task.
    -The most closest task to it's due date comes at top of the list.
    *Allow users to see tasks as depends on the deadlines*

5. History list
    -Users will see the history of completed and deleted task below.
